行、列、单元格

x    100
y    200
z    300
dtype: int64

此时只是定义了一个序列，并没有定义行还是列。只有当添加到一个DataFrame时才是距离的行和列。

   A   B    C
1  1  10  100
2  2  20  200
3  3  30  300

以字典的方式定义的数据帧是按列排列的，这也是最标准的定义方式

     1    2    3
A    1    2    3
B   10   20   30
C  100  200  300

以列表的方式定义的数据帧是按行排列的，注意区别

     A     B      D
1  1.0  10.0    NaN
2  2.0  20.0  100.0
3  3.0  30.0  200.0
4  NaN   NaN  300.0
import pandas as pd
from datetime import date, timedelta

def add_month(d, md):
    yd = md//12
    m = d.month + md % 12
    if m != 12:
        yd += m // 12
        m = m % 12
    return date(d.year + yd, m , d.day)
# 将数据读入books;并且跳过前3行;使用C到F列;数据帧中ID列的数据类型设置为整型(如果直接设置为整型会报错，所以设置成字符串类型)
books = pd.read_excel('C:/Temp/Book1.xlsx', skiprows=3, usecols='C:F', dtype={'ID': str, 'inStore': str, 'Date': str})
start = date(2018, 1, 1)  # 设置初始日期
print(books)  # 打印数据帧books
print(books['ID'])  # 打印books中的ID列
print(type(books['ID']))  # 打印ID列的属性
books['ID'].at[0] = 100  # ID列中0索引处设置为100
print(books['ID'])

for i in books.index:
    books['ID'].at[i] = i + 1
    books['inStore'].at[i] = 'Yes' if i%2 == 0 else 'No'  # 在Python中if...else...除了组成语句之外还会组成表达表达式。
    books['Date'].at[i] = add_month(start, i)
books.set_index('ID', inplace=True)
print(books)
books.to_excel('C:/Temp/Output.xlsx')
